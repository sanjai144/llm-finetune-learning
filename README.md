# llm-finetune-learning
# LLM Fine-tuning Learning Project

Fine-tuning TinyLlama on custom datasets to learn AI/ML fundamentals.

## 🎯 Goals
- Learn data cleaning & preparation (pandas)
- Understand LLM architecture & tokenization
- Practice efficient fine-tuning with LoRA
- Build a strong portfolio project for college admissions

## 📊 Current Status
🚀 **Phase 1: Project Setup** ✅
